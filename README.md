# Pedra, Papel e Tesoura
Este projeto é uma versão digital do clássico Pedra, Papel e Tesoura, desenvolvida com HTML, CSS, JavaScript, PHP e Bootstrap.
A proposta é trazer o tradicional jogo de escolhas rápidas para o ambiente web, com uma interface moderna, responsiva e fácil de usar.

O jogador enfrenta o computador em rodadas ágeis e divertidas, com resultados imediatos, placar atualizado e uma experiência visual agradável. Cada partida combina sorte e estratégia, desafiando o jogador a tentar prever a próxima jogada da máquina.

Principais recursos:

Escolha entre Pedra, Papel ou Tesoura

Sistema de jogadas automáticas do computador

Indicação clara de vitória, derrota ou empate

Contador de pontos e botão para reiniciar a partida

Layout responsivo utilizando Bootstrap

Integração em PHP para gerenciamento da lógica e comunicação entre as partes do jogo

Desenvolvido com atenção aos detalhes e foco na experiência do usuário, este projeto combina simplicidade, interatividade e design moderno.
Um clássico repaginado para a web — feito para jogar, se divertir e desafiar o computador em qualquer lugar!
